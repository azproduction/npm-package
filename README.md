# npm-package

This is an npm package stub.

Creating new package:

```
curl https://raw.github.com/azproduction/npm-package/master/install.sh | sh
```

Includes:

 * Readme.
 * License.
 * Contributing guide.
 * Changelog.
 * `jshint` config.
 * `jscs` config.
 * `csscomb` config.
 * `mocha` testing.
 * Test coverage calculation.
 * Travis integration.
 * Base Makefile.
 * Base `.gitignore`/`.npmignore`.
